Getting Started

This webpage is still under construction. Check back later for more info! 
sudo macgem install hotcocoa
hotcocoa myApp
cd myApp && rake run

HotCocoa is built on top of Cocoa, so your best friends will be the Cocoa documentation and HotCocoa's own documentation.

Contribute

Contributing to HotCocoa is fun and easy.
